# 🏫 School Management Dashboard

An interactive School Management Dashboard built with **Python** and **Streamlit** for managing and analyzing student records.

---

## 📌 Features

- 🔐 Login Authentication
- 📊 Interactive Dashboard
- 📌 KPI Cards
- 🔎 Student Search using GR Number
- 👨‍🎓 Student Information Display
- 📚 Subject-wise Marks Analysis
- 📈 Gender Distribution
- 💰 Fee Status Analysis
- 📖 Subject-wise Class Average
- 🎯 Grade, Section & Gender Filters
- 🚪 Logout Functionality

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Pillow (PIL)
- OpenPyXL
- Excel Dataset
