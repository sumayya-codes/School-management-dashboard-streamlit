
# 🏫 School Management Dashboard

An interactive School Management Dashboard built with **Python** and **Streamlit** for managing and analyzing student records.

---

## 📌 Features

- 🔐 Login Authentication
- 📊 Interactive Dashboard
- 📌 KPI Cards
- 🔎 Student Search using GR Number
- 👨‍🎓 Student Information Display
- 📚 Subject-wise Marks Analysis
- 📈 Gender Distribution
- 💰 Fee Status Analysis
- 📖 Subject-wise Class Average
- 🎯 Grade, Section & Gender Filters
- 🚪 Logout Functionality

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Pillow (PIL)
- OpenPyXL
- Excel Dataset

## 📊 Dashboard Modules

- Secure login using username and password.

- Male Students
- Female Students
- Average Obtained Marks

### 🔍 Student Search
Search students using their GR Number and view:
- Student Details
- Attendance
- Subject-wise Marks
- Fee Status
- Progress

### 📈 Charts & Analytics
- Gender Distribution
- Fee Status Analysis
- Subject-wise Marks Chart
- Subject-wise Class Average
